
# Watcher Protocol

Constitutional governance engine for AI memory.

**The problem:** AI systems have no evidence requirements. They generate confident output with no anchor to verified fact. This is not a hallucination problem. It is an infrastructure problem.

**The solution:** The Watcher Protocol is the missing governance layer — the evidence requirement that should exist underneath every AI system before output is trusted.

Like SSL for data integrity. Like DNS for name resolution. The Watcher Protocol routes claims to evidence.
